Functions

C functions aren't as simple as some think. The way a beginner programmer would make one is by creating it inside
of the main function of the program, however this isn't the most proper way to do it.

The way to create a proper function is by making it outside of the main function, then calling it into the main function.
The first way this is done is via "Call by Value". This way takes the intial value and makes copies of it but does not
overall change that value.

/* function definition to swap the values */
void swap(int x, int y)
{ int temp;
temp = x; /* save the value of x */
x = y; /* put y into x */
y = temp; /* put temp into y */
return; }

After "Call by Value" The other important method for functions is call by reference. "Call by Reference" takes the original
value, and modifies it instead of making a copy. This is generally the harder action to perform and is not used as much as
"Call by Value". You will notice the use of the "*". This is used to reference which variables need to be ran through the function.

/* function definition to swap the values */
void swap(int *x, int *y)
{ int temp;
temp = *x; /* save the value at address x */
*x = *y; /* put y into x */
*y = temp; /* put temp into y */
return; }

Arrays

A huge role in almost all programming languages including C, arrays are a piece of programming everyone should learn.

type arrayName [ arraySize ];

An array is a kind of data structure that can store a fixed-size sequential collection of elements of the same type. Its used
to store a collection of data, but it is often more useful to think of an array as a collection of variables of the same type.
The output of an array will typically look like a list, and will usually start at 0.

If Else Statements

If else statements open up alot of doors to give programs logic. Logic lets programmers keep their programs from making
errors as well as opening up other options for users to make.

if(boolean_expression) {
/* statement(s) will execute if the boolean expression is true */
}
else { /* statement(s) will execute if the boolean expression is false */
}

The "If Else" statement is seen in every other language in their own form of the function. withouht it, programs would have
many holes and issues.

/* local variable definition */
int a = 100;
/* check the boolean condition */
if( a < 20 ) {
